## 🚚 Logistics API - NestJS Challenge

Resolución del checkpoint para Backend Developer. Esta API gestiona un sistema de logística (Usuarios, Camiones, Órdenes y Ubicaciones) utilizando una arquitectura modular y robusta.

## 🛠️ Stack Tecnológico
* Framework: NestJS (Node.js)
* Lenguaje: TypeScript
* Base de Datos: MongoDB con Mongoose
* Autenticación: Passport.js & JWT
* Integración: Google Places API (Axios/HttpModule)


## 🏗️ Arquitectura y Abordaje
Decidí utilizar NestJS porque provee una estructura altamente organizada y escalable desde el inicio.
